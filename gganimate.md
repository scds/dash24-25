# Visualizing with R gganimate

This virtual workshop will introduce gganimate in the statistical programming language R. This workshop will go examples on how to create visualizations that change over time and how to and customize the design of those graphs. This will build on the previous workshop Data Visualization in R using ggplot2 which is recorded on the Sherman Centre website.
